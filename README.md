
# ☕ Starbucks Beverage Analytics Dashboard


### 1. ☕ Starbucks Beverage Analytics Dashboard: Brewing Insights, Fueling Connections

An interactive Power BI dashboard designed to analyze Starbucks beverage nutrition data, helping users explore calorie, sugar, protein, and caffeine trends across different beverage categories.

---

## 2. Short Description

The Starbucks Beverage Analytics Dashboard provides a comprehensive analysis of Starbucks drink offerings through interactive visualizations and KPI tracking. It enables users to compare beverage categories based on nutritional metrics such as calories, sugar, protein, and caffeine while uncovering insights into category performance and product composition.

---

## 3. Tech Stack

The dashboard was built using the following tools and technologies:

* 📊 **Power BI Desktop** – Main platform used for dashboard development and interactive reporting.
* 📂 **Power Query** – Used for data cleaning, transformation, and preparation.
* 🧠 **DAX (Data Analysis Expressions)** – Used to create KPIs, calculated measures, and dynamic filtering.
* 🔗 **Data Modeling** – Structured beverage and nutritional datasets for efficient analysis.
* 📈 **Data Visualization** – KPI Cards, Area Charts, Bar Charts, Donut Charts, Slicers, and Custom Visual Design.
* 📁 **File Formats** – `.pbix`, `.csv`, and dashboard screenshots.

---

## 4. Data Source

### Source: Starbucks Beverage Nutrition Dataset

The dataset contains nutritional information for Starbucks beverages, including:

* Beverage Category
* Beverage Preparation Type
* Calories
* Sugar
* Protein
* Caffeine
* Beverage Names

The data was cleaned and transformed using Power Query before visualization.

---

## 5. Features / Highlights

### • Business Problem

Consumers today are increasingly conscious about nutritional intake, yet comparing beverage options based on calories, sugar, caffeine, and protein can be difficult.

Starbucks offers a diverse beverage portfolio, making it challenging to quickly identify:

* Which drinks contain the highest caffeine?
* Which categories have the highest calorie content?
* Which beverages are suitable for health-conscious customers?
* How are beverages distributed across categories?
* What nutritional trends exist across Starbucks products?

---

### • Goal of the Dashboard

The dashboard aims to:

* Analyze Starbucks beverages based on nutritional content.
* Compare beverage categories using key health metrics.
* Identify high-caffeine and high-calorie products.
* Support customer decision-making through transparent nutritional insights.
* Provide an engaging and interactive analytical experience.

---

### • Walkthrough of Key Visuals

#### KPI Overview Cards

The dashboard presents four key performance indicators:

* Total Beverage Categories: 16
* Average Sugar: 30.37 g
* Average Calories: 213.92 kcal
* Average Caffeine: 77 mg

These KPIs provide an instant overview of the Starbucks beverage portfolio.

---

#### Protein Range Filter

An interactive slider allows users to filter beverages based on protein content.

**Business Value:**
Helps identify protein-rich beverages and supports personalized nutritional analysis.

---

#### Beverage Preparation Filter

A dynamic slicer enables users to analyze beverages by preparation type.

**Business Value:**
Allows comparison of nutritional characteristics across different beverage preparations.

---

#### Average Calories by Beverage Category

An area chart visualizes average calorie levels across beverage categories.

**Key Insight:**
Smoothies and Signature Espresso Drinks contribute the highest calorie levels, while Shaken Iced Beverages contain relatively fewer calories.

---

#### Beverage Category Distribution

A donut chart displays the percentage share of each beverage category within the dataset.

**Key Findings:**

* Classic Espresso Drinks: 31%
* Tazo® Tea Drinks: 25%
* Signature Espresso Drinks: 19%
* Smoothies: 19%
* Shaken Iced Beverages: 6%

This visualization highlights category dominance within Starbucks' beverage lineup.

---

#### Average Caffeine by Category

A horizontal bar chart compares average caffeine content across beverage categories.

**Key Insight:**

* Classic Espresso Drinks have the highest caffeine levels.
* Shaken Iced Beverages follow closely.
* Smoothies contain minimal caffeine.

This helps consumers identify beverages based on desired energy levels.

---

#### Top 5 Highest Caffeine Beverages

A featured visual showcases beverages with the highest caffeine content:

* Basic Coffee – 294 mg
* Classic Espresso – 122 mg
* Frappuccino Blended – 102 mg
* Frappuccino Light – 100 mg
* Iced Beverages – 93 mg

**Business Value:**
Identifies Starbucks' strongest caffeine offerings and helps understand product positioning.

---

### • Business Impact & Insights

#### Nutritional Awareness

Customers can easily compare beverages based on calories, sugar, protein, and caffeine.

#### Product Portfolio Analysis

Identify beverage categories that contribute the most to calorie and caffeine consumption.

#### Health-Conscious Decision Making

Supports consumers seeking lower-calorie, lower-sugar, or higher-protein beverage options.

#### Marketing Opportunities

Highlights popular beverage segments that can be promoted to specific customer groups.

#### Data-Driven Product Strategy

Provides insights that can support menu optimization and future product development initiatives.
